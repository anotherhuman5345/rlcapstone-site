Leukemia cell classifier — live demo

Select a real blood-smear cell below. The model classifies it into one of four classes — Benign or one of three B-cell ALL stages — in your browser. Nothing is uploaded.

Before you begin — this cannot diagnose leukemia.
  • Diagnosing leukemia requires a hematologist, bone-marrow analysis, flow cytometry, and genetic testing — never a single cell image.
  • This is a demonstration of image classification on a research dataset, and (as the write-up explains) its headline accuracy is inflated by how the data is split. Treat every output as an educational example.
  • If you have any health concern, consult a qualified clinician.

Select a cell

Each thumbnail is a real peripheral-blood-smear image of one white cell from the held-out test set.

On this dataset the model appears near-perfect — which is precisely what warrants suspicion. The write-up explains why, and what the patient-split v2 will reveal.

What am I looking at?

Each image is a single white blood cell, stained and photographed under a microscope. The malignant classes (Early Pre-B, Pre-B, Pro-B) are maturation stages of the same B-cell leukemia; the benign class is a non-cancerous look-alike. Distinguishing them is subtle even for specialists — which makes it a substantive computer-vision problem, and which is why honest evaluation matters so much.
