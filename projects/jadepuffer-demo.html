Catching AI-driven database ransomware

The flip side of my AI hacking agent — this one defends. It reads a database's activity and flags JADEPUFFER, the first ransomware ever run entirely by an AI. Watch it scan a log line by line.

Defensive and for learning. This only reads a log and scores it — it never runs anything and never touches a real database.

Researchers at Sysdig found the first ransomware attack driven start-to-finish by an AI: it breaks into a database, makes a secret admin account, encrypts everything, deletes the originals, and leaves a Bitcoin ransom note. My tool watches for exactly those moves.

Why this is the v2

V1 was an AI that attacks a practice site. As real AI-run attacks like JADEPUFFER start showing up, the more useful skill is catching them — so v2 flips the whole idea around into a detector.
